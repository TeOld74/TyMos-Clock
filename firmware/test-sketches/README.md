# Test Sketches

This folder will contain Arduino sketches for testing individual components and subsystems.

## Planned Test Sketches
- I2C scanner and PCA9685 test
- Individual servo channel test
- RTC (DS3231) test
- WiFi connection test
- NTP synchronization test
- Button and joystick input test
- Full movement sequence test
- Calibration helper sketch

## Usage
These sketches are useful for:
- Initial hardware verification
- Troubleshooting individual components
- Calibration procedures
- Learning how each component works
