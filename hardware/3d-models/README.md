# 3D Models

This folder will contain 3D printable parts and mechanical designs for the TyMos Clock.

## Planned Content
- STL files for 3D printing
- CAD source files (Fusion 360, SolidWorks, etc.)
- Assembly instructions
- Bill of materials (BOM)

## Components
- Segment frames
- Servo mounts
- Base structure
- Control panel enclosure
